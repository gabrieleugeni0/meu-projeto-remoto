# Introducao ao GIT

## Primeiros passos

- Como inicializar o projeto (repositorio local):

```bash
$ git init
```

- Como descobrir o status do repositorio:

```bash
$ git status
```

- Como adicionar novas mudancas:

```bash
$ git add <NOMEDOARQUIVO>
```

- Como gravar as mudancas:

```bash
$ git commit -m "MESSAGE"
```
